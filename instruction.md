# Makefile commands

## *Activating the virtual environment*
```bash
make activate
```

## *Running the django server*
```bash
make runserver
```

## *Making migrations to database*
```bash
make migrations
```

## *Migrating to database*
```bash
make migrate
```

## *Adding dependencies or packages*
```bash
poetry add <package name>
```
